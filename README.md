## Usage

Make your custom cell extend LTSwipeTableViewCell and call the corresponding methods

```objective-c
@interface LTSwipeTableViewCell : UITableViewCell
-(void) addRightView:(UIView *) view;
-(void) animateBack;
@end
```

See the example project for details~
